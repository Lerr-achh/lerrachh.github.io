HTML - язык гипертекстовой разметки

Файлы. html открываются бразузером.

Справочник тегов.

  • Теги для текста
  • Теги для выделения текста
  • Теги для графических элементов
  • Теги, задающие структуру страницы

1.Теги для текстов

  • Тег p от англ. paragraph - абзац
  • h1..h6 от англ. heading - заголовки
  • ol/ul-маркерованные и нумерованные списки от англ. unordered/ordered list
  • li-от англ. list item -элемент списка
  • br- перенос строки
  • a - от англ. anchor(якорь) ссылка

    2. Теги для выделения текса

    • b - жирный
    • i - курсивный
    • del - подчеркнутый
    • cite -цитата
    • tt - стиль печатной машинки

    3. Теги для графических элементов

    • hr - линия от horizontal rule

    4.Теги, задающие структуру страницы

    • !DOCTYPE html - не тег, а инструкция для браузера
    • html - заключает в себе все содержимое веб страницы
    • head - для хранения служебной информации о странице
    • body - содержимое документа HTML
    • section - задаёт раздел документа
    • div - универсальный контейнер
